Projeto desenvolvido em parceria com a Escola Estadual Presidente Tancredo Neves.

O objetivo inicial era criar um robô totalmente autônomo. Após discussões com os organizadores, optamos por um modelo semiautônomo, inspirado no funcionamento de robôs industriais.

Fui responsável pela programação do robô, implementando um sistema no qual ele executa comandos iniciais e, em seguida, repete os movimentos previamente realizados — uma lógica similar a processos automatizados em linhas de produção.

O robô também foi projetado para participar de atividades e desafios propostos pela escola, demonstrando resistência e versatilidade.
