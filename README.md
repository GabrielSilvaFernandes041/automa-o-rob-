Projeto desenvolvido em parceria com a **Escola Estadual Presidente Tancredo Neves**.

A ideia inicial era criar um robô totalmente autônomo, mas após conversas com os organizadores, optamos por um modelo **semiautônomo,** inspirado no funcionamento de robôs industriais.

Fui responsável pela parte de **programação**, implementando um sistema onde o robô executa comandos iniciais e, em seguida, **repete os movimentos previamente realizados** uma lógica similar a processos automatizados de linha de produção.

O robô também foi projetado para participar de atividades e desafios propostos pela escola, demonstrando resistência e versatilidade.
